# CSV TO ARRAY

CSV FILE TO ARRAY.

## Install

Via Composer

``` bash
$ composer require frankphper/csvtoarray
```

## Usage

``` php
$arr = new \Frankphper\Csv\csvToArray\csvToArray($csvFile);
print_r($arr->csvToArray());
```
